# AI Mini Apps with Gemini CLI

## Overview

This project demonstrates the use of **GitHub Copilot**, **VS Code**, and **Gemini CLI** to build AI-powered command-line applications using natural language prompts. The applications were created by describing requirements in plain English and allowing AI-assisted tools to generate the code.

## Features

* AI-assisted development with GitHub Copilot
* Command-line applications built using Gemini CLI
* Simple and beginner-friendly project structure
* Interactive text-based applications
* Practical examples of Generative AI workflows

## Applications

### 1. Poetry / Joke Generator

Generate a short 5-line poem or a joke based on a selected mood or theme.

### 2. Explain Like I'm 5

Simplifies difficult topics and explains them in easy-to-understand language.

### 3. Movie / Cricket Trivia Bot

Answer questions about movies, actors, cricket teams, players, and related trivia.

### 4. Study Notes Summarizer

Converts lengthy notes into concise bullet points for quick revision.
